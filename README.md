# Tunistratups

A full‑stack web platform for the Tunisian startup ecosystem.  
Backend: Flask + SQLAlchemy + JWT  
Frontend: React (CRA)

## Features
- User registration/login (JWT)
- Startup workspaces (owner + members via join code)
- Posts, comments, reactions, notifications
- Tasks with assignment and status
- Contract simulation with signatures
- ML‑based startup scoring
- FX rates and legal/loans hub

## Tech Stack
**Backend:** Flask, Flask‑Smorest, Flask‑SQLAlchemy, Flask‑JWT‑Extended, Marshmallow, Flask‑Migrate  
**Frontend:** React, React Router, Axios  
**ML:** scikit‑learn, pandas, joblib  
**DB:** MySQL (pymysql)
